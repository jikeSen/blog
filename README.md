#### 安装扩展 推荐使用yarn （安装时间有点长）
```
cnpm install
```
```
yarn (推荐使用)
```

#### 安装ui组件和客户端请求组件 (必须安装)
```
cnpm install iview axios sass-loader node-sass sass-graph --save

```
#### 启动
```
npm run dev
```
#### 构建生产打包
```
npm run build
```
